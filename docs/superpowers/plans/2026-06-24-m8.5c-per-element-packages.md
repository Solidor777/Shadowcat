**Goal:** Complete M8.5 by splitting the monolithic `core-ui` into per-element
in-game packages under `src/modules/*`, moving the layout into a module behind a
`root` surface (so the shell hard-codes no region), relocating `scene-tools`, and

**Architecture:** Final slice (a, b done). Seam-enabling refactors first (asset
REST → `@shadowcat/core`; AppContext `logout`; module-owned layout via a `root`
surface; `WorldSession` `modules: Module[]`), then the mechanical package
extraction. Pure refactor — **zero behavior change** (binary e2e is the proof).

**Tech Stack:** pnpm workspaces, Vite 8, Svelte 5 (runes), Vitest 4 + jsdom +
`@testing-library/svelte`, TypeScript 5.6.

## Global Constraints

- **No behavior change.** Mechanical extraction + seam plumbing. The binary-served
  Playwright e2e (`src/client/shell/e2e/*.spec.ts`, 9 tests) must stay green
  unedited — it exercises the canvas, tool rail, panels, and entry flow.
- **`@shadowcat/core` stays Svelte-free.** Asset REST (plain `fetch`) is fine in
  core; no Svelte import enters core's closure.
- **Contract-only element boundary.** Each in-game package communicates ONLY via
  ui-kit seams (AppContext, contributions, `<Surface>`) + core/render/types —
  never by importing another module or the shell. The shell imports the modules
  (one direction); a module importing the shell would cycle.
- **Module-owned layout.** The shell hosts a single `root` surface; the layout
  module owns the region grid. The shell hard-codes no region contract.
- **`dist/` path + CI ordering unchanged.** Only `@shadowcat/shell` builds to `dist/`.
- **Whole-tree stale-ref sweeps** (no `--include` allowlist) after each move.
- **Refactor-shaped cycle:** move/plumb → typecheck/test/build green → commit.

```ts
import type { Asset } from "@shadowcat/types";

/** Upload an image to a world; returns the created asset record. */
export async function uploadAsset(world: string, file: File): Promise<Asset> {
  const form = new FormData();
  form.append("file", file);
  const res = await fetch(`/api/worlds/${world}/assets`, { method: "POST", body: form });
  if (!res.ok) throw new Error(`upload failed: ${res.status}`);
  return (await res.json()) as Asset;
}

/** List a world's assets (the grid source). */
export async function listAssets(world: string): Promise<Asset[]> {
  const res = await fetch(`/api/worlds/${world}/assets`);
  if (!res.ok) throw new Error(`list failed: ${res.status}`);
  return (await res.json()) as Asset[];
}

/** Replace an asset's bytes behind its stable UUID; returns the updated record. */
export async function replaceAsset(uuid: string, file: File): Promise<Asset> {
  const form = new FormData();
  form.append("file", file);
  const res = await fetch(`/api/assets/${uuid}/replace`, { method: "POST", body: form });
  if (!res.ok) throw new Error(`replace failed: ${res.status}`);
  return (await res.json()) as Asset;
}

/** Delete an asset (file + record). */
export async function deleteAsset(uuid: string): Promise<void> {
  const res = await fetch(`/api/assets/${uuid}`, { method: "DELETE" });
  if (!res.ok) throw new Error(`delete failed: ${res.status}`);
}
```
